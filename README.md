# The Tomato C Library

**Tomato** is a C library (libc) developed for [**the Radish Operating System** (RadishOS)](https://github.com/wither16x/radish_os).

## Features
- Basic crt0
- 3 headers

## Types
### [`stddef.h`](include/stddef.h)
- `size_t`
### [`stdint.h`](include/stdint.h)
- `uint8_t`
- `uint16_t`
- `uint32_t`
- `uint64_t`
- `int8_t`
- `int16_t`
- `int32_t`
- `int64_t`
